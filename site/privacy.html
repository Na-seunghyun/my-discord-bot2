Privacy

This page explains what data the Kingshot Command Hub uses and what it never asks for.

Data Used by the Site

  • Auto Redeem may store Kingshot player IDs, public profile summaries, gift-code job status, and recent redemption result messages.
  • Feedback may store the message, optional contact text, related page, browser user-agent, and request IP for abuse prevention and troubleshooting.
  • Visitor counting is limited and used only to show lightweight site usage statistics.
  • Calculator and map planner data is mostly processed in the browser.

Data Not Requested

  • No Kingshot password or game account login.
  • No email password, payment card, bank account, or private device access.
  • No executable download is required to use the web tools.

Removal Requests

If a player ID was registered by mistake, use the Auto Redeem removal request or send a feedback message with the player ID to review.
