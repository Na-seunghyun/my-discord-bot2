Security

This site is served over HTTPS through Cloudflare Workers. The tools are designed to be transparent: no password, payment information, email login, or device access is requested.

HTTPS No password collection No payment collection Limited API writes

What We Do

  • Use security headers, including HTTPS/HSTS, frame blocking, content-type protection, and a strict content security policy.
  • Block common scanner paths such as hidden repository files, environment files, WordPress probes, and server configuration files.
  • Limit cross-site write requests and reject oversized or invalid JSON API requests.
  • Keep admin actions behind an admin token configured in Cloudflare secrets.

What We Do Not Ask For

  • No game password or account login.
  • No email password, device permission, browser extension, or app installation.
  • No credit card or payment information.
  • No executable file download is required to use the website.

Report a Security Issue

Please report suspicious behavior, incorrect redirects, unexpected downloads, or privacy concerns through the feedback page.
